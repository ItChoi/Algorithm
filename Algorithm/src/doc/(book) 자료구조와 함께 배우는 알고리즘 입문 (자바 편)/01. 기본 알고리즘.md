Q1) 네 값의 최대 값을 구하는 max4 메서드를 작성하세요.

	A1)
	```class
	public static int max4(int a, int b, int c, int d) {
		int max = a;
		
		if (b > max) {
			max = b;
		} 
		
		if (c > max) {
			max = c;
		} 
		
		if (d > max) {
			max = d;
		}
		
		return max;
	}
	}
	```

	- 각 조건을 부정하고 논리곱을 논리합으로, 논리합을 논리곱으로 바꾸고 다시 전체를 부정하면 원래의 조건과 같다.
		ex) x && y와 !(!x || !y)는 같다.
