## Java 해쉬(Hash) 기본 개념과 구조

	1) 해쉬(Hash)의 기본 개념
		- 해쉬는 내부적으로 배열을 사용하여 데이터를 저장하기 때문에 빠른 검색 속도를 갖는다.
		   그리고 데이터의 삽입과 삭제 시 기존 데이터를 밀어내거나 채우는 작업이 필요 없도록 특별한 알고리즘을
		   이용하여 데이터와 연관된 고유한 숫자를 만들어 낸 뒤 이를 인덱스로 사용한다.
		   따라서 삽입과 삭제 시 데이터의 이동이 없도록 만들어진 구조이다.
		   
		   해쉬가 내부적으로 사용하는 배열을 Hash Table이라고 하며 그 크기에 따라 성능 차이가 많이 날 수 있다.
